For easy map creation, I recommend using Notepad++ with this User-Defined Language.

To add the NINMAP language to Notepad++ follow these steps:
0. Choose whether you want a light or dark theme
1. In Notepad++, Language-> Define your Language-> Import
	and select "/* theme/userDefinedLang_NINMAP.xml"
2. Copy NINMAP.xml to your Notepad++ API folder
	find it at "\Program Files(x86)/Notepad++/plugins/APIs"
	or something like that
3. Exit and re-open Notepad++
4. You are now able to use the NINMAP language.

(Optional). In Settings->Style Configurator, you may need to "Enable global background colour".


Tips for Notepad++:
CTRL+Space		: Function autocomplete box
CTRL+SHIFT+Space	: Function Parameter tip
